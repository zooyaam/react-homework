## 📎 구현 내용 정리

---

- 사용되는 아이콘 및 이미지 파일 → public > assets 폴더에 생성
- Header, Main, Footer 별로 컴포넌트 분리 → src > components 폴더에 생성
- 각 컴포넌트를 App.jsx에 연결 → App.jsx를 main.jsx에 연결하여 웹 브라우저에 렌더링
